Create a QR code on your website with this nice block.

== Description ==

This plugin registers a editor block that allows implement a QR code image easily.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/qr-block` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
